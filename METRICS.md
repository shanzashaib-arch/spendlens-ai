# Metrics

## North Star Metric

Completed audits per week.

This metric directly measures whether users find enough value to complete the audit flow.

## Input Metrics

### Landing Page Conversion Rate
Percentage of visitors who begin an audit.

### Audit Completion Rate
Percentage of started audits successfully completed.

### Share Rate
Percentage of audits shared through public URLs.

## Initial Instrumentation

I would first track:
- page visits
- audit submissions
- share button clicks
- lead capture conversions

## Pivot Trigger

If audit completion rate remained below 20% after several iterations, I would reconsider onboarding complexity and perceived value proposition.
